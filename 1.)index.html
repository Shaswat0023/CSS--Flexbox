CSS Flexbox

Flexbox is a great way to get more flexiblity in your layouts and to simplify responsive layout design. It makes it easy to align elements on a 2D plane easy to use once you get familiar with the main properties.

    1. The first step is to set display: flex on a container element. the childern to the flex container become flex items. A set of properties can be applied to flex containers and heave an effect to all the items as a whole, and a different set of properties can be applied to flex items and have their effect on the targeted items. Flex items can in turn also be flex containers for the elements it contains, making it easy to create complex layouts.
